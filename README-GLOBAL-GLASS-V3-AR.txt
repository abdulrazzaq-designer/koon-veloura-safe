Veloura Global Glass V3 — Neutral Frosted

التعديل:
- زجاج أبيض/فضي محايد بدون درجة زرقاء.
- تقليل تشبع لون الخلفية خلف الزجاج إلى 36% حتى لا يكتسب لون المتجر الأحمر أو الأزرق.
- خفض البلور إلى 17px ورفع شفافية السطح مقارنة بـ V2، لتظهر العناصر خلفه بصورة أوضح.
- إضافة حلقة حافة محايدة 1px فقط، بلا ظل منتشر، حتى لا يندمج الزجاج مع الخلفية البيضاء.
- تسجيل الدخول: البلور على الصندوق الداخلي فقط.
- الـ overlay والـ host والـ wrapper بكامل الشاشة: تعتيم خفيف فقط، بدون blur وبدون filter.
- إصلاح CSS المحقون داخل Shadow DOM الذي كان يضع البلور على s-modal-wrapper و role=dialog.
