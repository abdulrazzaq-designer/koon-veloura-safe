حزمة Koon Veloura — V3 Glass + Sticky Fix

التعديلات:
- زجاج بلوري محايد موحد للهيدر والتبويبات والقائمة العائمة والقائمة الجانبية.
- لمسة اللون 3% إلى 7% فقط؛ اللون الظاهر يأتي من خلفية الصفحة مثل أسلوب تيليجرام.
- الهيدر والتبويبات يأخذان نفس عرض الكونتينر ونفس المسافة الجانبية.
- فاصل صغير ثابت بين الهيدر والتبويبات.
- نفس الحواف المختارة للهيدر تُطبق على التبويبات.
- عند اختفاء الهيدر تصبح التبويبات أول سطح أعلى الشاشة دون فراغ ارتفاع الهيدر.
- إصلاح زر تثبيت الهيدر: تحويل القيمة إلى Boolean حقيقي، وإعادة قياس الارتفاع، وتثبيت الهيدر مباشرة بعد تجاوزه.
- نفس المسافة العلوية للهيدر تُستخدم أسفل القائمة العائمة عند تفعيل البلور.

التركيب:
فك الحزمة داخل جذر المشروع مع الاستبدال، ثم pnpm production وبعده salla theme preview.
